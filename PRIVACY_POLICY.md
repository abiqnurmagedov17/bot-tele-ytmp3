# Privacy Policy for YouTube Downloader Bot

**Last Updated:** 16 April 2026

## 1. Data Collection
This bot **does not collect or store any personal data**. We do not log:
- User messages or chat history
- Downloaded video/audio links
- Personal information (name, phone number, email)

## 2. How the Bot Works
- The bot only processes YouTube links sent by users
- Download links are generated via third-party public APIs (scraped)
- No user data is shared with any third party

## 3. Third-Party Services
This bot uses ytmp3.mobi public API for video conversion. Please refer to their privacy policy for data handling on their end.

## 4. Data Retention
No data is retained. All processing happens in-memory and is discarded immediately after the download link is sent.
